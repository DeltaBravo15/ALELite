Making A Call

The overall point to this software is to allow the user to place a call to another station and exchange messages. This is the option to do that.

Before a call can be placed, however, you need to open an existing configuration file or create one so that the program has a valid Self Address and a list of Other Addresses to which you can place a call.

To begin sounding, pull down the "Transmit" menu from the toolbar and select one of the following,

  • Allcall
  • Individual Call
  • AMD Message
