Testing

Get the ALE Clean Tones CDROM from ITS. The disk contains many sample ALE tones you can use to test your setup.

To test your setup,

  • Insert the ALE Tones CD into your CD drive.
  • Run any program that is designed to play audio CDs.
  • Call up your volume control display (double click on the speaker icon in your system tray). Select "Options" and then "Properties." Select "Recording" and "Ok".
    • If your CD drive is internal to the computer, make sure there is a check mark under "CD" and none of the other input devices. This setting now means that only sounds from the internal CD player will flow into the system. Note: The CD cable that came with your sound card must connect the output of the CD to the CD input of the sound card.
    • If your CD player is external to the computer (such would be the case if you are using an external stereo system), make sure the check mark is under "Line In" and none of the other input devices.
  • Run ALE Lite and then begin playing the CD. The tones on the disk will simulate ALE transmissions from other stations.
