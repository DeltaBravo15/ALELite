Saving The Configuration File

Whenever you have edited your configuration, you will want to save the changes. To do this, pull down the "File" menu from the tool bar and select "Save" (to resave the file using the same name) or "Save As" (to save the file under a new name).

Also, whenever you exit the program normally (i.e., you select "Exit" from the toolbar), the program will automatically save your configuration for you under the last configuration file name you selected.
