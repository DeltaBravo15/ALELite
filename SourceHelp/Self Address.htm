Self Address

You will need to create an address that other stations will use to call your station.

To add to or edit your Self Address, pull down the "Edit" menu from the toolbar and select "Self Address."

Into the "Self Address" box, type in a user name that identifies your station. Note: The first entry in the Self Address list must be exactly 3 characters long. If the first Address you enter isn't 3 characters, or you try to move a longer Address to be the first entry in the list, you will receive an error message.

Into the "Net Address" box, type in the name of the network to which you might want to make a Network Call. Note: Network Call is not implemented in this version of the software, so any value entered in this box is currently ignored.

Type in a whole number into the "Slot Wait" for the number of slots to wait when receiving a Network Call. Note: Network Call is not implemented in this version of the software, so any value entered in this box is currently ignored.

Select, from the list on the right side of the window, the frequencies over which you will be using this Self Address (the selections will be highlighted as you choose them).

When you have selected all the required information, click on the "Ok" button.

Follow the same instructions when editing a Self Address.

Your changes to the Self Address list will be saved when you save your configuration file.
