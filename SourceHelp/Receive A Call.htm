Receiving A Call

The other overall point to this software is to allow the user to configure their system to continuously scan selected channels for messages with its Self Address. This is the option to do that.

Before calls can be received, however, you need to open an existing configuration file or create one so that the program has a valid list of Self Addresses to which your station can receive calls.

To begin scanning for calls, pull down the "Receive Mode" menu from the toolbar and select "Scanning" option. The program will begin scanning all channels associated with the Self Address(es) in your configuration file.

Note: The associated "Valid Channels" selections you made in the Self-Adress configuration steps will be used during scanning to determine on which channels to listen for which Self Addresses.

Alternately, you can select "Single Channel" from the "Receive Mode" drop-down menu to scan on one single channel for any Self Address(es) associated with that channel. You will be asked to specify which channel you want to scan after you select this option.
