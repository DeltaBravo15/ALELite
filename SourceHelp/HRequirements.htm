Hardware Requirements

The computer you install the software and sound card into should be moderately powerful. The minimum system that the software has been tested on is,
  • Pentium - 200 Mhz
The sound card you use must,
  • use DirectSound protocols.
Microsoft displays a list of sound-card manufacturers (some are Direct-Sound combatible, some are not) and their phone numbers that can be contacted for more information.
Currently, the only type of radio that this program supports is,
  • ICOM.
