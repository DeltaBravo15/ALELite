DLL Files

In order for this program to run, several DLL files must exist in your Windows system directory (these include the Direct Sound files plus a few others required by the ALE Lite software. All of DLL files that you will be using were produced by Microsoft. Many of the DLL files already exist in most typical Windows configurations and can be found in the \Windows\System\ subdirectory on your hard drive.)

Important: The files included in the \DLL directory on this CD ROM should be copied to your computer only if they do not exist or are replacing older versions of these files on your computer. If newer versions of these files already exist on your computer, do not copy the files from this CD ROM onto your hard drive. Use your existing files instead.
