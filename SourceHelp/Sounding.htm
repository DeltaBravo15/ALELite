Sounding

An ALE radio station needs to, periodically, announce its presence to the rest of the stations that are listening (scanning).

This sounding allows all of the stations to monitor the quality of the channels between it and the sounding station. This quality information is stored in a database and then consulted when an actual call is placed between two stations.

Before sounding, you need to open an existing configuration file or create one so that the program has a valid list of your Self Addresses.

To begin sounding, pull down the "Sounding" menu from the toolbar and select "Start Sounding." You will be presented with a list of the Self Addresses in your configuration file. Select the Selef Addresses you would like to sound on and then click on the "Ok" button.

To stop scanning, select the "Sounding" drop-down menu and select the "Stop Scanning" option.

Note: The sounding intervals and channels that you specified in the Self-Adress configuration steps will be used during this sounding process to determine how often to sound on that address.
