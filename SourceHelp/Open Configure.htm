Open the Configuration File

To open a configuration file, select "File" from the toolbar menu and select "Open." Find the configuration file you want (configuration files have the ".ale" extension) and open it by double-clicking on the name or by pressing the "Open" button below the selection.

Note: If this is the first time you have run the program, there will be no configuration files to open. You need to edit one and then save one for future use.
