Edit The Configuration File

Once the configuration file has been opened, select "Edit" from the pull-down menu, and then select "Configuration" as the item to edit.

Now, you can edit any of the following items which will determine how the various computers communicate with each other an with each radio,

  • Radio Type.
  • Radio Address.
  • Serial Communications Port.
  • Baud Rate.
Once all changes have been made, you should save the file. If you don't, the program will save the changes for you when you exit the program. If the program ends abnormally, and you haven't yet saved your changes, those changes will be lost.
